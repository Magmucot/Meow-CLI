# Meow CLI VS Code Extension

This extension brings the power of Meow CLI directly into VS Code.

## Features

- **Terminal AI Assistant**: Chat with Gemini or other models directly from your sidebar.
- **Website-like Design**: A sleek, terminal-inspired UI.
- **Integrated Workflow**: Access AI assistance without leaving your editor.

## Getting Started

1. Install the extension.
2. Open the Meow CLI view from the Activity Bar.
3. Start chatting!
